# Why Should we use Redis?

Because it is incredebly fast, it is not fast because of it's architecture or it is written in some specific language
It is fast because of :-

- Using RAM for storage
- Organising data in **traditional data structure**
- Not have too many features like other databases like *postgresql, mysql*
